## 1. 设计语言

### 1.1 Claymorphism 粘土风格
整体设计采用 **Claymorphism（粘土风格）** 设计语言，特点是：
- **暖色调** — 以米白 `#fffbf8` 为背景主色
- **柔和阴影** — 暖灰色阴影（rgba(230, 198, 179, opacity)）
- **大圆角** — 卡片圆角 36rpx~48rpx
- **哑光质感** — 无高饱和渐变色，一切以柔和为主

### 1.2 设计原则
1. **温暖感** — 适合亲子类应用的情感基调
2. **亲和力** — 圆润、柔软、无尖锐边缘
3. **简洁但精致** — 去除冗余装饰，保留精心设计的细节（阴影/圆角/间距）

## 3. 阴影系统

所有阴影使用 **暖灰色**（rgba(230, 198, 179, ...)），不透明度按层级递增：

| 层级 | 阴影值 | 用途 |
|------|--------|------|
| 最弱 | `0px 2px 4px 0px rgba(230,198,179,0.2)` | 非活跃筛选标签 |
| 弱 | `0px 3px 8px 0px rgba(230,198,179,0.18)` | 字段卡片、上传选项 |
| 中 | `0px 6px 16px 0px rgba(230,198,179,0.35)` | 宝宝信息卡片、用户卡片 |
| 强 | `0px 6px 20px 0px rgba(230,198,179,0.25)` | 上传卡片 |

**品牌色阴影**（橙色主色调专用）：
| 用途 | 阴影值 |
|------|--------|
| 橙色按钮 | `0px 4px 8px 0px rgba(255,168,122,0.4)` |
| 筛选活跃态 | `0px 3px 6px 0px rgba(255,168,122,0.35)` |
| 保存按钮 | `0px 3px 6px 0px rgba(255,168,122,0.3)` |

## 6. 布局规范

### 6.1 设计稿宽度
- **375px** 设计稿宽度，对应 **750rpx** 小程序宽度
- 转换公式：`1px = 2rpx`

### 6.2 页面结构
所有页面遵循统一框架：
```
┌─ StatusBar (safeTop高度, 由系统决定) ─┐
├─ Navigation Bar (88-104rpx) ──────────┤
├─ Content Area (flex:1, scroll-y) ─────┤
│   (各页面内容)                         │
├─ Bottom Nav (128rpx, fixed) ──────────┤
```

### 6.3 间距
- 页面左右边距：40rpx
- 卡片间距：20rpx
- 字段行间距：20rpx
- 筛选标签间距：16rpx

## 7. 组件设计规范

### 7.1 卡片 (Card)
```
┌──────────────────────┐
│  ┌─────┐            │
│  │ 📷  │  图片占位   │
│  └─────┘            │
│  第一次翻身          │

```
- 宽度：324rpx（2列）
- 圆角：36rpx
- Padding：28rpx
- 4种颜色交替使用

### 7.2 底部导航 (Bottom Nav)
```
┌──────┬──────┬──────┬──────┐
│  🏠  │  📖  │  ➕  │  👤  │
│ 首页  │ 相册  │ 上传  │ 我的  │
└──────┴──────┴──────┴──────┘
```
- 活跃态：`#ffa87a` + 半粗字重
- 非活跃态：`#999` + 中字重

### 7.3 筛选标签 (Filter Chip)
- 活跃态：橙色背景 + 白色文字 + 橙色阴影
- 非活跃态：白色背景 + 灰色文字 + 浅灰色阴影

### 7.4 字段行 (Field Row)
```
┌─────────────────────────────┐
│ [图标] 标签名称    [输入区域] │
└─────────────────────────────┘
```
- 高度：128rpx
- 左侧：图标(72rpx) + 标签
- 右侧：flex:1 弹性间隔
- 尾部：输入框/选择器/切换区域

### 8.4 首页 Header 动画 (Collapsible Baby Header)
**展开态（默认）**：
```
┌──────────────────────┐
│  ┌────┐              │
│  │ 👶 │ 小星星 ✨    │
│  │    │ 6个月3天·女宝│
│  └────┘          ⚙  │
└──────────────────────┘
```

**折叠态（上划触发）**：
```
┌──────────────────────┐
│ 👶  👧  👦  ...  ➕  │  ← 水平头像条
└──────────────────────┘
```
- 折叠动画：内容区开始滚动时，Header 卡片向上收起为水平条
- 头像列表：水平排列所有宝宝圆形头像（40px），可左右滑动
- 末尾固定：橙色加号(+)按钮，点击跳转宝宝列表页
- 轮盘切换：点击折叠态中的宝宝头像 → 切换 currentBabyId → Header 更新为该宝宝信息并展示展开态
- 全局变量：`currentBabyId` 存于 storage，首页所有内容跟随切换
