## 页面描述

用户选择照片/视频后，在上传的同时弹出编辑浮层，填写媒体信息。

## 布局

### 多文件切换

- 浮层底部有「上一张 / 下一张」按钮
- 当前编辑第 N / 共 M 张
- 切换时自动保存当前编辑内容

## 需要输出的 Figma

1. 上传编辑浮层完整设计（Bottom Sheet 样式）
2. 位置选择区域（自动获取 + 手动 + 最近使用）
3. 标签选择器（预置标签列表 + 自定义输入）
4. 美好时刻文本输入框（带字数限制提示）
5. 里程碑选择器（图文并茂的横向滚动选择）
6. 多文件切换（上一张/下一张 + 进度指示）
7. 全部保存按钮
8. 上传进度条（半透明区域）
