## 组件用途

选择照片对应宝宝的发展阶段，帮助家长标记宝宝的成长里程碑。

## 组件设计

### 横向滚动选择

一行横向可滚动的卡片，每张卡片代表一个阶段：

```
┌──────┐ ┌──────┐ ┌──────┐ ┌──────┐ ┌──────┐ ┌──────┐ ┌──────┐ ┌──────┐
│ 🍼   │ │ 👶   │ │ 👐   │ │ 🦷   │ │ 🚶   │ │ 🏃   │ │ 🗣️  │ │ 🎨   │
│新生儿 │ │抬头期│ │探索期│ │出牙期│ │学步期│ │行走期│ │语言期│ │创造期│
│0-1月  │ │1-3月 │ │3-6月 │ │6-9月 │ │9-12月│ │12-18 │ │18-24 │ │24-36 │
└──────┘ └──────┘ └──────┘ └──────┘ └──────┘ └──────┘ └──────┘ └──────┘
```

### 选中态

选中时卡片放大（scale 1.05）+ 橙色边框 + 阴影。

### 宝宝月龄联动

- 自动根据当前宝宝月龄推荐阶段
- 例如 6 个月宝宝默认选中「探索期」
- 用户可以手动调整

## 里程碑阶段定义

| 图标 | 名称 | 月龄 | 关键词 |
|------|------|------|--------|
| 🍼 | 新生儿 | 0-1月 | 出生、满月、黄疸消退 |
| 👶 | 抬头期 | 1-3月 | 抬头、微笑、追视、咿呀 |
| 👐 | 探索期 | 3-6月 | 翻身、抓握、坐立、啃咬 |
| 🦷 | 出牙期 | 6-9月 | 出牙、爬行、咿呀学语 |
| 🚶 | 学步期 | 9-12月 | 站立、扶走、第一句话 |
| 🏃 | 行走期 | 12-18月 | 独立行走、奔跑、模仿 |
| 🗣️ | 语言期 | 18-24月 | 词汇爆发、短句、自我意识 |
| 🎨 | 创造期 | 24-36月 | 画画、角色扮演、社交 |

## 需要输出的 Figma

1. 里程碑选择器完整设计（横向滚动卡片式）
2. 选中态（橙色边框 + 放大）
3. 各阶段的图标 + 名称 + 月龄
4. 根据月龄自动推荐的提示
