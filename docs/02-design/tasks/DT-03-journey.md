## 页面描述

对外可见的宝宝成长时间轴。按月分组展示已归档的媒体，带位置、标签、时刻描述、里程碑信息。可分享给家人查看。

## 页面布局

### 顶部导航栏

```
👶 小星星的成长历程            [分享]
```

- 标题含宝宝头像和昵称（动态）
- 右侧橙色分享按钮

### 宝宝 Header（顶部信息卡）

```
┌─────────────────────────────────┐
│  ┌────┐                         │
│  │ 👶 │  小星星 ✨              │
│  │    │  1岁2个月 · 128张记录    │
│  └────┘                         │
│                                 │
│  🍼新生儿 → 👶抬头期 → 👐探索期   │
│  ←——●————●————●——————→        │
│    满月  百日  翻身  当前        │
└─────────────────────────────────┘
```

- 左侧头像 + 右侧昵称 + 年龄 + 记录总数
- 下方里程碑进度条

### 时间轴分组

按月分组，每组包含标题和媒体卡片：

```
🎉 2026年6月（12张）
┌──────────────────────────────────┐
│  ┌──────────┐ ┌──────────┐       │
│  │ 📷 第一次 │ │ 📷 北京   │       │
│  │  翻身 🎉 │ │  公园    │       │
│  │ 📍 家里   │ │ 📍 朝阳   │       │
│  │ ✨ 宝宝第 │ │ 🏷️ 外出   │       │
│  │  一次... │ │          │       │
│  │ 🍼翻身期  │ │          │       │
│  └──────────┘ └──────────┘       │
└──────────────────────────────────┘

┌──────────────────────────────────┐
│  ┌──────────┐ ┌──────────┐       │
│  │ 📷 满月   │ │ 📷 第一   │       │
│  │  照      │ │  次笑    │       │
│  └──────────┘ └──────────┘       │
└──────────────────────────────────┘
```

### 媒体卡片（含信息）

每张卡片展示：
- 缩略图
- 标题/时刻描述（限 1 行）
- 位置图标 + 地名
- 标签（首个小标签）
- 里程碑图标（如果有）

### 对外分享

点击分享按钮 → 生成微信小程序分享卡片：
- 标题：「小星星的成长历程」
- 图片：封面图（用户可选择）
- 路径：`/pages/journey/journey?shared=true`
- 被分享者只能看到已归档内容

### 空状态（无归档时）

```
┌─────────────────────┐

│  在「素材库」中      │
│  归档后展示在这里    │
│                     │
│  [去素材库看看]     │
│                     │
└─────────────────────┘
```

## 交互说明

| 操作 | 行为 |
|------|------|
| 上下滚动 | 按月份分组的时间轴滚动 |
| 点击卡片 | 跳转全屏查看器 |
| 点击分享 | 对外分享成长历程 |
| 点击里程碑进度条 | 筛选对应阶段媒体 |
| 下拉刷新 | 重新加载 |
