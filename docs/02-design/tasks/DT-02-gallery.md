## 页面描述

未归档媒体的批量管理中心。所有刚上传的媒体默认进入素材库，用户在此归档、编辑、筛选后发布到成长历程。

## 页面布局

### 顶部导航栏

```
← 返回      📦 素材库        [批量选择]
```

- 左侧返回箭头的圆形背景按钮（同 v1.x 风格）
- 标题「素材库」
- 右侧「批量选择」文字按钮（橙色）

### 筛选栏（Chip 行）

水平滚动的筛选标签：

```
[全部] [未归档] [已归档] [标签 ▼] [位置 ▼] [月龄 ▼]
```

- 全部：默认选中
- 未归档/已归档：归档状态筛选
- 标签/位置/月龄：点开下拉菜单
- Chip 样式：活跃态橙色底白字，非活跃态白底灰字

### 媒体网格（主要区域）

2 列瀑布流网格，每张卡片展示：

```
┌──────────┐
│  📷      │
│  缩略图   │
│          │
├──────────┤
│ 📍 北京  │
│ 🏷️ 第6月 │
│ ✨ 第一次 │
└──────────┘
```

- 长按进入批量选择模式
- 选择模式下：卡片左上角出现复选框
- 非选择模式：点击进入全屏查看器

### 批量操作栏（选择模式时出现）

底部固定栏：

```
[全选] [标签] [归档] [删除]
```

- 半透明背景，固定在底部导航上方
- 显示已选数量：「已选择 3 项」
- 操作按钮禁用态 → 选中后高亮

### 空状态

```
┌─────────────────────┐
│                     │
│     📦              │
│  素材库是空的        │
│  上传的照片和视频    │
│  会出现在这里        │
│                     │
│  [+ 上传新内容]     │
│                     │
└─────────────────────┘
```

## 交互说明

| 操作 | 行为 |
|------|------|
| 点击卡片（非选择模式） | 跳转全屏查看器 |
| 长按卡片 | 进入选择模式，卡片出现复选框 |
| 点击复选框 | 选中/取消选中 |
| 点击「全选」 | 选中当前筛选结果全部 |
| 点击「归档」 | 所选媒体标记为已归档，移入成长历程 |
| 点击「标签」 | 弹出标签选择器 |
| 点击「删除」 | 弹出确认框 → 软删除 |
| 下拉刷新 | 重新加载媒体列表 |
| 上拉加载 | 分页加载更多 |
