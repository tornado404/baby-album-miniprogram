# 产品需求文档 (PRD)

## 宝宝成长相册 — 微信小程序

> 版本：v1.1 | 最后更新：2026-06-04

---

## 1. 产品概述

### 1.1 产品定位
**宝宝成长相册** 是一款面向新手父母（尤其是90后/00后）的微信小程序，帮助家长记录、整理和回顾宝宝成长的每一个珍贵瞬间。以照片/视频为核心载体，结合年龄里程碑标注，打造有温度的亲子成长日记。

### 1.2 目标用户
- **核心用户**：0-3岁宝宝的父母/监护人
- **次级用户**：祖父母、亲戚等家庭成员（通过共享查看）
- **用户画像**：忙碌但注重仪式感的年轻父母，习惯使用微信生态

### 1.3 核心价值
1. **记录** — 快速上传宝宝照片/视频，自动关联年龄信息
2. **回顾** — 时间轴浏览，按年龄段筛选，回忆成长轨迹
3. **分享** — 方便地分享给家人好友（二期规划）

## 3. 用户旅程

### 3.1 首次使用流程
```
打开小程序 → 登录引导页(Onboarding) → 微信一键登录 →
宝宝信息填写(昵称+头像) → 保存 → 跳转首页(Album Home)
```

### 3.2 渐进式引导流程
```
首次登录填写昵称和头像后 →
进入首页开始使用 →
后续在设置页可通过"宝宝管理"进入完整档案补充信息
（性别、出生日期、预产期、体重、身高等）
```

### 3.3 日常使用流程
```
打开小程序 → 浏览时间轴 → 筛选月份 → 点击卡片查看详情 →
上传新照片 → 编辑宝宝档案 → 查看统计
```

## 4. 非功能需求

### 4.1 性能指标
- 首页首屏加载时间 < 3秒
- 图片列表渲染 < 1秒（100张以内）
- 页面切换响应 < 500ms

### 4.2 兼容性
- 微信基础库版本 >= 3.0.0
- iOS 12+ / Android 8+
- 适配全面屏（安全区域）

### 4.3 安全
- 用户数据仅存储在本地（一期）
- 不上传未加密的个人信息
