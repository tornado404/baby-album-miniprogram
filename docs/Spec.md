## 术语对照

| 英文 | 中文 | 说明 |
|------|------|------|
| PRD | 产品需求文档 | 定义功能和用户故事 |
| Epic | 大特性 | 一组相关功能的集合 |
| P0 | 最高优先级 | 核心功能，必须完成 |
| P1 | 高优先级 | 重要功能，应尽快完成 |
| P2 | 中优先级 | 扩展功能，按需排期 |
| Claymorphism | 粘土风格 | 柔和/圆润的设计语言 |
| Skyline | 天空线 | 微信新一代渲染引擎 |
| WebView | 网页视图 | 传统渲染引擎 |
