# DT-01: 底部导航 Tab 重构

> 优先级：P0 | 关联 PRD：第 2 章

## 当前状态

底部导航 4 个 Tab：🏠 首页 / 📖 相册 / ➕ 上传 / 👤 我的
其中「首页」和「相册」路由到同一页面。

## 设计要求

### Tab 项变更

| 原 Tab | 新 Tab | 图标 | 路由 | 说明 |
|--------|--------|------|------|------|
| 🏠 首页 | 👶 成长历程 | 宝宝图标 | `/pages/journey/journey` | 已归档媒体，对外可见 |
| 📖 相册 | 📦 素材库 | 箱子图标 | `/pages/gallery/gallery` | 未归档媒体，仅自己可见 |
| ➕ 上传 | ➕ 上传 | 加号 | `/pages/upload/upload` | 不变 |
| 👤 我的 | 👤 我的 | 用户 | `/pages/settings/settings` | 不变 |

### 视觉规范

- 活跃态：橙色 `#ffa87a` + 图标填充 + 文字 SemiBold
- 非活跃态：灰色 `#999` + 图标线框 + 文字 Regular
- 上传 Tab 居中突出：橙色圆形背景 + 白色「+」
- 高度：128rpx（含安全区域）
