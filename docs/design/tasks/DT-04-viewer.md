# DT-04: 全屏媒体查看器

> 优先级：P0 | 关联 PRD：第 4 章 | 路由：`/pages/media_detail/media_detail`

## 当前问题

当前 media_detail 页面使用 Mock 数据，无真实图片展示，无手势交互，无信息展示。

## 页面布局

### 暗色背景全屏

整个页面为纯黑背景 `#000000`，媒体内容居中。

### 顶部栏（半透明叠加）

```
← 返回      📷 3/12          ••• 菜单
```

- 半透明黑色渐变背景（顶部渐隐）
- 左侧返回按钮（白色）
- 中间媒体序号（「3/12」）
- 右侧菜单按钮

### 媒体主体

- 图片：居中显示，保持宽高比，最大尺寸填充
- 视频：自动播放，显示播放/暂停控制
- 支持双指缩放（pinch zoom）
- 支持左右滑动切换上一张/下一张
- 滑动时有平滑 transition

### 底部信息面板（半透明叠加）

```
┌─────────────────────────────────┐
│ 📍 北京市朝阳区    🏷️ 第6月 微笑  │
│ ✨ 今天宝宝第一次自己翻身了！     │
│ 🍼 里程碑: 翻身期                │
│ 📅 2026年6月6日 · 6个月3天       │
│                                 │
│  [🗑️ 删除] [📥 归档] [📤 分享]  │
└─────────────────────────────────┘
```

- 半透明黑色渐变背景（底部渐隐）
- 位置信息 + 标签展示
- 时刻描述文本（楷体/手写风格）
- 里程碑 + 日期
- 操作按钮行

### 信息面板交互

| 操作 | 行为 |
|------|------|
| 单击画面 | 切换显示/隐藏信息面板（顶部+底部） |
| 上滑 | 展开完整信息面板 |
| 下滑 | 收起信息面板 |
| 双指捏合 | 缩放图片 |
