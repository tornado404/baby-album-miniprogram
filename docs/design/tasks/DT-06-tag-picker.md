# DT-06: 标签选择器组件

> 优先级：P1 | 关联 PRD：第 3.3 章 | 使用页面：上传浮层、素材库批量标签

## 组件用途

预置标签展示 + 自定义标签输入，支持单选/多选，用于上传时标记媒体。

## 组件设计

### 标签分类 Tab

```
[月龄] [里程碑] [日常] [表情] [自定义]
```

- 横向滚动的 Tab 切换
- 活跃 Tab 下划线橙色

### 标签列表

每行显示 3 个标签：

```
┌──────┐ ┌──────┐ ┌──────┐
│ 🟣第1月│ │ 🟣第2月│ │ 🟣第3月│
└──────┘ └──────┘ └──────┘
┌──────┐ ┌──────┐ ┌──────┐
│ 🟠翻身│ │ 🟠坐立│ │ 🟠爬行│
└──────┘ └──────┘ └──────┘
```

- 选中态：填充色 + 白色文字
- 非选中态：线框 + 灰色文字
- 每个颜色类别不同底色

### 颜色系统

| 类别 | 选中色 | 文字色 |
|------|--------|--------|
| 月龄 | `#dceaf1` | `#2c6b8a` |
| 里程碑 | `#f1dce2` | `#8a2c4a` |
| 日常 | `#e2f1e6` | `#2a6b3a` |
| 表情 | `#f4e6d6` | `#8a6a2c` |
| 自定义 | `#e8e0f0` | `#5a3a8a` |

### 自定义输入

```
┌──────────────────────┬────┐
│ 输入新标签...         │添加│
└──────────────────────┴────┘
```

- 输入后点击「添加」或按回车
- 自动去重
- 最多 10 个标签/媒体
