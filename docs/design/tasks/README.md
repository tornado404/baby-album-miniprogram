## 设计规范

所有页面遵循 v1.x 的 Claymorphism 设计语言：
- 背景色：`#fffbf8`
- 主色：`#ffa87a`（橙色）
- 圆角系统：`12~48rpx`
- 阴影：暖灰色 `rgba(230,198,179, opacity)`
- 设计稿宽度：375px → 750rpx
