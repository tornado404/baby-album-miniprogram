# 宝宝日记导航重构 UI 设计规格

> **版本**: v1.0 | **最后更新**: 2026-06-17
> **状态**: 📝 设计阶段
> **配套**: [`baby-diary-navigation-product-analysis.md`](../02-design/baby-diary-navigation-product-analysis.md)（产品分析报告）

## 1. 设计目标

本次 UI 改版实现以下目标：

1. 底部导航重构为「成长 / 记录 / 我的」三 Tab
2. 成长页整合宝宝成长卡、年龄筛选、里程碑时间线、瀑布流媒体
3. 记录页支持多图片/视频上传，统一描述
4. 取消独立整理页，操作入口整合到成长页内容卡片（所见即所得）

## 2. 底部导航

### 2.1 Tab 配置

| Tab 键 | 名称 | 图标 | 页面映射 |
|--------|------|------|----------|
| growth | 成长 | 嫩芽/树苗 | album_home（改造） |
| record | 记录 | 相机/加号 | upload（改造） |
| profile | 我的 | 人物头像 | settings / baby_profile |

### 2.2 规范

- 活跃态颜色：`#ffa87a`（橙色）
- 非活跃态颜色：`#999999`（灰色）
- 底部导航组件位置：`miniprogram/components/bottom-nav/`

## 3. 成长页（album_home 改造）

### 3.1 布局结构

```
┌─────────────────────────────┐
│  宝宝成长卡                 │
├─────────────────────────────┤
│  年龄阶段筛选 chips         │
├─────────────────────────────┤
│  里程碑纵向时间线           │
├─────────────────────────────┤
│  瀑布流媒体卡片             │
│  （含操作按钮）             │
├─────────────────────────────┤
│  [+ 上传] FAB              │
└─────────────────────────────┘
```

### 3.2 宝宝成长卡

| 元素 | 说明 |
|------|------|
| 宝宝头像 | 圆形，48px，与宝宝档案头像一致 |
| 宝宝姓名 | 标题字体，16px 粗体 |
| 精确年龄 | 如「6 个月 15 天」 |
| 记录总数 | 如「共 128 条记录」 |
| 点击区域 | 整个卡片可点击，跳转宝宝档案 |

### 3.3 年龄阶段筛选

| 选项 | 说明 |
|------|------|
| 全部 | 显示所有记录 |
| 0-1月 | 0-30 天 |
| 2-3月 | 31-90 天 |
| 4-6月 | 91-180 天 |
| 7-12月 | 181-365 天 |
| 1岁+ | 366 天以上 |

- 选中态：高亮橙色背景
- 非选中态：灰色背景

### 3.4 里程碑纵向时间线

- 展示形式：纵向时间线，节点在左侧，内容在右侧
- 节点内容：里程碑图标 + 里程碑名称 + 日期
- 排序：倒序（最新在前）
- 折叠：默认展开最近 3 个，可展开/折叠
- 交互：点击节点筛选该里程碑所有记录

### 3.5 瀑布流媒体卡片

- 布局：双列瀑布流，倒序（最新在前）
- 卡片内容：
  - 缩略图（4:3 或根据原图自适应）
  - 日期（右下角）
  - 年龄标签（如「3 个月」）
  - 视频时长角标（视频专属）
- 操作按钮（⋮）：右上角，上传者=当前用户时显示

### 3.6 内容卡片操作按钮

点击 ⋮ 展开操作菜单：

| 操作 | 说明 |
|------|------|
| 设为精选 | 添加到成长页精选流 |
| 隐藏 | 从成长页隐藏，不删除 |
| 删除 | 彻底删除，需二次确认弹窗 |

**权限规则**：仅上传者本人可见操作按钮。

### 3.7 空状态

| 场景 | 显示内容 |
|------|----------|
| 无宝宝 | 引导创建宝宝档案 |
| 有宝宝无记录 | 引导上传第一张照片 |

### 3.8 右下角 FAB

- 位置：页面右下角固定
- 图标：+ 或相机
- 点击：跳转记录页

## 4. 记录页（upload 改造）

### 4.1 Step 1: 选择来源

三个入口按钮，Claymorphism 风格：

| 入口 | 图标 | 触发动作 |
|------|------|----------|
| 拍照 | 📷 | 拉起相机 |
| 相册 | 🖼️ | 打开相册多选 |
| 视频 | 🎬 | 打开相册多选视频 |

### 4.2 Step 2: 确认信息

```
┌─────────────────────────────┐
│  ← 添加记录                │
├─────────────────────────────┤
│  已选择 N 个文件           │
├─────────────────────────────┤
│  宝宝    [选择 ▼]           │
├─────────────────────────────┤
│  日期    [2026-06-17 ▼]    │
├─────────────────────────────┤
│  描述    [这一刻...]       │
├─────────────────────────────┤
│  里程碑  [选择 ▼/跳过]     │
├─────────────────────────────┤
│      [ 确认记录 ]           │
└─────────────────────────────┘
```

- 已选数量提示
- 宝宝选择：下拉选择已有宝宝
- 日期：默认当天，可修改
- 描述：一条描述对应所有已选素材
- 里程碑：可选，跳过则不标记

### 4.3 Step 3: 上传中

```
┌─────────────────────────────┐
│  上传中...                 │
├─────────────────────────────┤
│  ┌─────────────────────┐   │
│  │ ████████░░ 80%      │   │
│  └─────────────────────┘   │
│                             │
│       第 N 张 / 共 M 张      │
│                             │
│  [ 取消 ]         [重试]   │
└─────────────────────────────┘
```

- 进度条显示百分比
- 当前第几张 / 总数
- 失败显示重试按钮

### 4.4 Step 4: 完成反馈

```
┌─────────────────────────────┐
│           ✅                │
│                             │
│     已加入成长相册           │
│                             │
│  ┌─────────────────────┐   │
│  │   预览刚才的记录     │   │
│  └─────────────────────┘   │
│                             │
│  [查看记录]    [继续上传]    │
└─────────────────────────────┘
```

- 展示刚上传的内容缩略图
- 按钮：查看记录（跳转详情）/ 继续上传（重置流程）

## 5. 我的页面

保持现有功能不变：
- 宝宝档案入口
- 家人共享入口
- 设置入口

## 6. 权限与数据规则

| 规则 | 说明 |
|------|------|
| 操作按钮可见性 | 仅上传者本人可见内容卡片操作按钮 |
| 家人共享 | 管理员可管理所有内容，普通成员仅可管理自己上传的内容 |
| 默认展示 | 上传后内容默认出现在成长页，无需额外整理步骤 |
| 精选内容 | 设为精选后进入精选流，普通用户可见 |

## 7. 设计规范

### 7.1 Claymorphism 风格

- 背景色：`#fffbf8`（米白）
- 主色调：`#ffa87a`（橙色）
- 卡片背景：`#ffffff`
- 阴影：`0px 6px 16px 0px rgba(230, 198, 179, 0.35)`
- 圆角：16px-24px

### 7.2 交互规范

- 页面切换：使用微信小程序标准导航
- 按钮点击：50ms 内反馈
- 加载状态：显示骨架屏或 loading
- 错误提示：toast 提示

## 8. 实施优先级

| 优先级 | 模块 | 说明 |
|--------|------|------|
| P0 | 底部导航 | 改为 3 Tab |
| P0 | 成长页改造 | 宝宝卡 + 筛选 + 瀑布流 + 操作按钮 |
| P0 | 记录页完整流程 | 选择 → 确认 → 上传 → 反馈 |
| P1 | 里程碑时间线 | 纵向展示 + 折叠 |
| P1 | 多素材上传 | 相册多选 + 统一描述 |
| P2 | 家人权限精细化 | 管理员 vs 普通成员 |

## 9. 验收标准

- [ ] 底部导航正确显示 3 个 Tab，切换逻辑正确
- [ ] 成长页显示宝宝信息、年龄筛选、瀑布流
- [ ] 内容卡片操作按钮仅上传者可见
- [ ] 记录页可多选图片/视频，统一点击描述上传
- [ ] 上传流程完整：选择 → 确认 → 进度 → 完成反馈
- [ ] 我的页面功能保持不变
