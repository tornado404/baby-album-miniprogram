# 相册功能 Task 调整方案

**文档版本**: v1.0
**创建日期**: 2026-06-01
**功能模块**: 宝宝成长相册 (ALBUM_V1)
**状态**: 待评审

---

## 1. 背景

设计文档 (`docs/feature/album_v1/feature_design.md`) 与实际代码实现存在较大差距。采用**方案B：实现向设计看齐**，以设计文档为基准，修复实现中的差距。

### 1.1 差距分析

| 设计需求 | 当前实现状态 | 差距描述 |
|----------|--------------|----------|
| FR-01 相册首页 - 动态标题 | 固定标题"成长相册" | 标题应显示"宝宝姓名+的成长相册" |
| FR-01 相册首页 - 分页加载 | 只有 Toast 提示 | 需要完整的分页加载实现 |
| FR-02 时间线导航 - 月龄标签 | 范围筛选"0-3月" | 应为具体月龄"0月"、"1月"... |
| FR-02 时间线导航 - 动态生成 | 固定选项 | 应根据宝宝月龄动态生成 |
| FR-03 图片上传 - FAB 菜单 | 直接选择图片 | 应有悬浮菜单（拍照/相册/视频） |
| FR-03 图片上传 - 进度提示 | 无 | 应显示上传进度 |
| FR-04 瀑布流展示 - 组件集成 | masonry_layout 未使用 | 首页直接用 block 渲染 |
| FR-05 媒体详情 - 双指缩放 | 未实现 | 应支持图片缩放 |
| FR-06 月龄分组 - MediaGroup | 未实现 | 缺少分组数据结构 |

### T-07-REV: 月龄筛选功能修复

**修复内容**:

**验收标准**:
- [ ] 标签显示具体月龄"0月"、"1月"...

---

### T-08-REV: 本地存储服务修复

**修复内容**:
1. 添加 `MediaGroup` 数据结构
2. 实现 `getMediaGroupedByMonthAge()` 方法
3. 支持按月龄分组查询

**数据结构**:
```typescript
interface MediaGroup {
  monthAge: number;      // 月龄（0, 1, 2, ...）
  monthLabel: string;    // 显示标签，如 "0月"、"1月"、"12月+"
  mediaList: Media[];    // 该月龄的媒体列表
  mediaCount: number;    // 该月龄的照片数量
}
```

**验收标准**:
- [ ] MediaGroup 数据结构定义正确
- [ ] getMediaGroupedByMonthAge() 方法可用
- [ ] 分组数据按时间倒序
